Behavior System - Heavily In Progress

The main idea is to use a stack that maintains and manages behaviors. The bottom element in the stack is the default behavior, and whenever interesting things happen that would cause a new behavior, that behavior is pushed to the top of the stack. When that behavior is finished, it pops, and the default behavior is resumed.
